# **App Name**: NaviLoop Tracker

## Core Features:

- Bus Simulation: Simulate the bus location using a Python script and push the coordinates to Firebase Realtime Database.
- Breakdown Detection: Detect and transmit bus breakdown status to Firebase Realtime Database.
- ETA Calculation: Calculate ETA based on distance and average speed. Consider adding functionality to simulate traffic.
- Breakdown Alert: Visually alert students on the frontend when a bus breakdown is detected.
- Real-time Map Visualization: Display the live bus location and student location on a map using React and Leaflet.js.
- User Authentication: Implement user authentication using Firebase Authentication to manage student logins.

## Style Guidelines:

- Primary color: Blue (#2E9AFE) to evoke trust, security, and reliability in the bus tracking service.
- Background color: Light gray (#F0F4F8) for a clean and modern interface.
- Accent color: Orange (#FFA500) to highlight important alerts such as ETA updates and bus breakdowns.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for both headlines and body text.
- Use simple, clear icons to represent bus stops, bus locations, and user locations on the map.
- Use a responsive layout that adapts to both desktop and mobile devices, ensuring a consistent user experience.
- Incorporate subtle animations for bus movements and ETA updates to provide real-time feedback.
